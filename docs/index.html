ODD SPACE

Introduction

Odd space is a RPG turn based game, where the protagonist gets lost from his space pirates crew, where he has been working as a kitchen boy.
Our protagonist ends up in an old and rusty ship with a crazy captain.
Equipped with a spatula and a pan, he lives some crazy adventures with the captain, fighting drunk people, exploring odd planets and finding treasures with zero value.

Features

Combat

  • Random enemy encounters
  • Turn-based combats
  • Enemies drop items

Dialogues

  • Branching system
  • Text appears letter by letter
  • Dialogues can affect the environment

Quests/Puzzles

  • Main quest
  • Different side quests
  • Dungeon with puzzles

Extra

  • Movement by tiles
  • Responsive UI
  • Animations and sound FX

Controls

Keyboard

  • ↑←↓→: Movement/Menu navigation
  • X: Interact/Dialogue skip
  • Z: Cancel
  • ESC: Open/Close pause menu

Gamepad

  • Joystick: Movement
  • D-pad: Menu navigation
  • A/X: Interact/Dialogue skip
  • B/O: Cancel
  • Start: Open/Close pause menu
